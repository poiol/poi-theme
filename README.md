## 功能
- 添加头图支持
- 列表添加预览/回退
- 调整展示样式

## 使用
1. 将本项目放入sharelist\theme目录
2. 修改本项目\script\config.js中头图与预览页背景色设置
3. 在sharelist后台选择poi-theme主题
4. 坐与放宽(
